Sign-to-Text Recognition Features

1. Real-Time Sign Language Recognition

Uses AI to detect and translate hand signs into letters (A-I) instantly.

2. Live Video Feed

Displays a real-time video stream with sign detection overlay.

3. Display Predicted Text

Shows the recognized sign dynamically on the screen.

4. User Controls

Includes Reset, Play (text-to-speech), and Stop buttons.

5. Text-to-Speech (TTS) Integration

Converts recognized text into speech for better accessibility

6. Flask Web Interface

Runs on a web browser with an easy-to-use interface

7. Optimized Performance

Uses OpenCV and MediaPipe for fast and accurate recognition.
